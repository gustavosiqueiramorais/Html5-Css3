Código fonte

A melhor maneira de exibirmos códigos dentro dos nossos sites em linguagem html é utilizando a tag Code, veja o exemplo abaixo de um código em python que recebe dois numeros e exibe a soma entre eles. 

    
    numero = int(input('Digite um numero inteiro: '))
    numero2 = int(input('Digite outro numero inteiro: '))
    soma = numero + numero2
    print('A soma dos dois numeros é igual a {}'.format(soma))
